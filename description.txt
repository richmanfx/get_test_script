
Программа "get_test_script" разработана на языке Python и предназначена 
для скрапинга тест-скриптов с серверов Мониторинга. После запуска программы 
достаточно выбрать требуемый сервер Мониторинга из списка и 
ввести номер тест-скрипта.

В файле конфигурации "get_test_script_cfg.py" предварительно указываются 
список серверов Мониторинга с TCP портами, имя и пароль пользователя на 
сервере, папка для сохранения тест-скриптов, браузер для скрапинга 
(PhantomJS, Chrome или Firefox) и, если требуется, размер окна браузера.

Программу можно запускать двумя способами: с помощью командного файла
"get_test_script.bat" (в операционной системе должен присутствовать
интерпретатор Python) или бинарным файлом "get_test_script.exe" из
подпапки "build/exe.win-amd64-2.7".

Файлы тест-скриптов сохраняются в кодировке UTF-8.
